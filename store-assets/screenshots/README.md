# bayLURE store screenshots

All screenshots are **9:16 portrait** (the aspect Google Play requires), PNG,
well under 8 MB, captured from the real running app. The web build carries no
ad banner (ads are native-only), so these are banner-free.

| Folder | Resolution | Play slot |
|---|---|---|
| `phone/` | 1080×1920 | Phone screenshots (2–8) |
| `tablet-7in/` | 1080×1920 | 7-inch tablet screenshots |
| `tablet-10in/` | 1440×2560 | 10-inch tablet (min side ≥1080) |

Each set is the same six screens in order: home + map, bite score, hourly +
conditions, lure picks, rigs & bait, catch log. Upload in numbered order.

Regenerate with `scratchpad/shots/shot2.js` against `npm run web` (localhost:8081).
